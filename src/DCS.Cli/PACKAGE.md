# Dependency Chain Substrate CLI

Install as a global .NET tool:

```bash
dotnet tool install --global DependencyChainSubstrate.Cli
```

Then run `dcs analyze`, `dcs diff`, `dcs fix`, and other commands from any directory.

Full documentation: https://github.com/tonythethompson/dependency-chain-substrate#readme

**Early access (0.1.x):** C# fix commands are preview-quality; Java is analyze-only. See [CHANGELOG.md](https://github.com/tonythethompson/dependency-chain-substrate/blob/main/CHANGELOG.md).
